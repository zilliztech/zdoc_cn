# 重排

混合搜索通过同时进行多个近似最近邻（ANN）搜索来实现更精确的搜索结果。多个搜索会返回多组结果，这就需要一种重排序策略来帮助合并和重新排序这些结果，并返回单组结果。本指南将介绍 Zilliz Cloud 支持的重排序策略，并提供选择合适重排序策略的建议。
