# Zilliz Cloud 跨集群迁移

Zilliz Cloud 跨集群迁移允许您在集群之间传输数据，无论它们位于同一组织内还是跨不同组织。主要使用场景包括：

- 从免费集群迁移到付费集群计划；

- 在不同组织之间安全共享数据；

- 重新组织基础设施以优化性能。
