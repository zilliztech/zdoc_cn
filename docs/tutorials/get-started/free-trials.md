# 免费试用

Zilliz Cloud 当前提供两种可免费试用的订阅版本：免费版和企业版。首次注册 Zilliz Cloud 时，您将获赠 ¥300 免费额度，有效期为 30 天，可用于试用 Zilliz Cloud 企业版。

## 免费版{#}

免费版适合想要尝试向量数据库的开发人员。

- **特性：**一个部署在阿里云上的免费 Serverless 集群。

- **定价：**免费，无需绑定任何支付方式。

- **配置：**无需额外配置即可快速开始。

- **能否免费试用：**可以。您可最多创建 2 个Collection。每个 Collection 容量约为 50 万 768维向量。

- **适合人群：**向量数据库领域新手或想要快速搭建 AI 应用的开发人员。

## 企业版{#}

企业版适合大型企业或组织，是构建向量数据平台首选方案。该方案提供高可用、数据安全、专家技术支持。

- **特性：**高级功能。提供全天候技术支持。

- **云服务提供商：**阿里云。

- **定价：**详见 [Zilliz Cloud 定价](https://zilliz.com.cn/pricing)

## 不同版本的免费试用对比{#}

|                      |  免费版         |  企业版                    |
| -------------------- | ------------ | ----------------------- |
|  集群类型                |  Serverless  |  Dedicated              |
|  试用有效期               |  永久          |  30 天                   |
|  适合用例                |  快速开始        |  验证性测试（PoC）             |
