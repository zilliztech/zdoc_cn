# 云服务提供商和地域

Zilliz Cloud 是一项基于公有云向量数据库集群服务。通过我们的服务，您可以在您选择的公共云平台上轻松创建和管理您自己的向量数据库集群。

Zilliz Cloud在阿里云、腾讯云和亚马逊云科技上提供跨区域的集群服务。您可以[随时联系我们要求增加新的云区域](https://zilliz.com.cn/cloud-region-request)。
