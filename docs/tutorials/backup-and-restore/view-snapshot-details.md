# 查看备份快照

本文介绍如何查看已创建的快照信息。您可以手动或自动创建快照。

## 快照状态{#snapshot-status}

根据不同场景，快照可能会处于如下状态：

- **创建中**
    如果快照处于此状态，则无法创建其他快照。但您可以单击**操作**列中的更多按钮，然后选择**取消**来终止创建过程。

- **成功**

如果您的快照状态处于其他状态，可能是出现了错误。如果该错误持续存在，请立即与我们联系。
