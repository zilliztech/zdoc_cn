# 支付方式与账单

了解订阅 Zilliz Cloud 服务的多种方式及发票开具的相关说明。

如果您的组织账号中同时具有优惠券、[现金余额](./advance-pay)和[云市场订阅](./subscribe-on-aliyun-marketplace)，则扣费顺序为：优惠券&gt;现金余额&gt;阿里云市场。

例如，当您有一份账单需要支付时，我们会先扣除您的优惠券。如果剩余优惠券不足以抵扣全部账单金额，我们会扣除所有剩余优惠券，剩余账单金额部分将扣除您的现金余额。如扣除优惠券和现金余额后，依旧不足以抵扣全部账单金额，我们将通过您绑定的阿里云云市场账号进行扣费。
