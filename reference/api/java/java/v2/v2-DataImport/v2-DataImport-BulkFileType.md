# BulkFileType

This is an enumeration that provides the following constants.

## Constants

- **PARQUET**

    Sets the file type to [Parquet](https://parquet.apache.org/) (*.parquet*).

- **JSON**

    Sets the file type to JSON (*.json*).

- **CSV**

    Sets the file type to CSV (*.csv*)
