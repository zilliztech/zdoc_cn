查看数据导入任务

列出指定集群上的数据导入任务。

GET
https://controller.api.{cloud-region}.cloud.zilliz.com.cn/v1/vector/collections/import/list

示例​

列出指定集群上的数据导入任务。

说明

此处请使用您的 API Key 做为 Token。

curl --request GET \
     --url "https://controller.api.${CLOUD_REGION_ID}.zillizcloud.com/v1/vector/collections/import/list?clusterId=${CLUSTERID}" \
     --header "Authorization: Bearer ${TOKEN}" \
     --header "accept: application/json" \
     --header "content-type: application/json" \

请求​

参数​

  • 查询参数

    参数名称参数说明
    clusterIdstring（必选）
    Zilliz Cloud 集群 ID。
    pageSizestring
    每次返回的导入任务数量。
    currentPagestring
    当前页码。

  • 路径参数

    参数名称参数说明
    CLOUD_REGION_IDstring（必选）

请求体​

无请求体。

响应​

返回一个导入任务列表。

响应体​

  • 处理请求成功后返回
{
    "code": 200,
    "data": {
        "count": "string",
        "currentPage": "string",
        "pageSize": "string",
        "records": [
            {
                "collectionName": "string",
                "jobId": "string",
                "state": "string"
            }
        ]
    }
}
  • 处理请求失败后返回
{
    "code": integer,
    "message": string
}

属性​

下表罗列了响应包含的所有属性。

属性名称属性描述
codeinteger
表示请求是否成功。
  • 200：请求成功。
  • 其它：存在错误。
dataobject
表示响应中携带的数据对象。
data.countstring
当前响应中包含的导入任务数量。
data.currentPagestring
当前页码。
data.pageSizestring
每次返回的最大记录数量。
data.recordsarray
data.records[].collectionNamestring
当前导入任务对应的目标 Collection 名称。
data.records[].jobIdstring
当前导入任务的 ID。
data.records[].statestring
当前导入任务的状态。可能的取值有： ImportRunning、ImportCompleted 和 ImportFailed.
messagestring
具体描述请示错误的原因。
