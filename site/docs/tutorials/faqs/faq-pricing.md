# 定价

本文列举了 Zilliz Cloud 定价相关的常见问题及对应解答。

**集群挂起后，是否还会收取费用？**

集群挂起后，我们仅收取存储费用，不会收取计算费用。更多详情，请阅读 [Zilliz Cloud 定价](https://zilliz.com.cn/pricing)。

**赠送的 ¥ 300 免费额度是否会过期？**

在首次注册后，您可获赠¥ 300 免费额度，有效期 30 天。

**Zilliz Cloud 如何处理不活跃集群？**

Zilliz Cloud 会自动挂起 7 天内不活跃集群，并发送邮件通知。您可随时按需恢复集群。

**如何避免未使用的集群产生费用？**

建议您挂起未使用的集群以节省成本。您可随时按需恢复集群。

**如何估算使用 Zilliz Cloud 的成本？**

请使用[价格计算器](https://zilliz.com.cn/pricing)估算使用成本。

**数据被移至回收站后是否收费？**

回收站中删除的数据不收取费用。

**预算有限的情况下，如何在使用 Zilliz Cloud 服务时节省成本？**

如果您的预算有限，我们推荐使用经济型 CU。该类型的 CU 容量与容量型 CU 相同，但价格更低。更多详情，请阅读[选择合适的 CU 类型](./choose-the-right-cu-type-and-size) 。
