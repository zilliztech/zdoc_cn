# 用户和角色

本文列举了 Zilliz Cloud 中用户和角色相关的常见问题及对应解决方法。

**是否可以离开组织？**

如果您是组织成员，您可以随意离开组织。

如果您是组织管理员，您只有在组织还有其他管理员的情况下离开组织。1 个组织必须拥有 1 名组织管理员。因此，如果您是组织中唯一一名管理员时，您不可离开组织。

**如何修改组织名称？**

1. 选择需要修改名称的组织。

1. 点击左侧导航栏中的**系统设置**。

1. 在系统设置页面下的**组织信息**板块中，点击**编辑**按钮。

1. 输入新的组织名称并点击**确认**按钮。

1. 界面右上角出现提示，您已成功修改组织名称。

**如何邀请同事或团队成员进行协作？**

如果您是组织管理员，请在左侧导航栏点击**成员**并邀请用户加入组织。

如果您是组织成员，请联系您所在组织的管理员邀请用户加入组织。

此外，Zilliz Cloud 还支持邀请用户加入项目。如果您是项目管理员，请点击左侧导航栏**项目成员**邀请用户加入项目。
