# 版本说明书 2.1.0

**发布日期：2023年8月21日**

我们很高兴地宣布Zilliz Cloud 2.1.0的发布。此版本引入了一系列增强和功能：

### RESTful API {#restful-api}

为了方便用户操作，我们开放了RESTful API接口。用户可以通过调用这些接口来操作 Zilliz Cloud 上的集群、Collection 及存放在 Collection 中的向量数据。欲了解更多，请前往 [API 参考](https://docs.zilliz.com.cn/reference/cloud-meta)。

## 阿里云云市场 {#alibaba-cloud-cloud-market}

为了方便用户订阅 Zilliz Cloud 的服务，我们开放了新的订阅渠道。现在用户可以通过阿里云云市场订阅我们的服务。欲了解更多，请前往[订阅阿里云市场](./subscribe-on-aliyun-marketplace)。
