# 免费试用

Zilliz Cloud 当前提供两种订阅版本：企业版和专有部署。

- **企业版**适合大型企业或组织，是构建向量数据平台首选方案。该方案提供高可用、数据安全、专家技术支持。您可直接在 Zilliz Cloud 上订阅企业版。

- **专有部署**适用于高度注重数据隐私和合规的场景。该版本无免费试用方案。如需部署，请[联系我们](https://zilliz.com.cn/contact-sales)。

首次注册 Zilliz Cloud 时，您将获赠 ¥300 免费额度，有效期为 30 天，可用于试用 Zilliz Cloud 企业版。
