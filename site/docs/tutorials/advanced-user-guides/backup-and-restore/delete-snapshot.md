# 删除备份快照

在快照达到指定保留天数后，Zilliz Cloud 将自动删除快照。您也可以在到达保留天数之前手动删除不需要的快照。

![delete_snapshot](/img/delete_snapshot.png)

在执行删除操作之前，系统会提示您确认是否删除快照。
