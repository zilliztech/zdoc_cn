# 角色与权限

Zilliz Cloud 提供访问控制。您可以灵活管理用户，为用户分配不同角色，授予用户不同的集群资源访问和操作权限。无权限的用户无法访问特定资源。

## 组织和项目 {#organization-and-projects}

Zilliz Cloud 通过组织和项目实现访问控制和资源分离。

组织是 Zilliz Cloud 中最高层级，组织中包含项目和成员。在大多数情况下，您可以根据公司或者其他分组来设置组织。

项目是组织层级下的概念，用于将集群及相关资源进行分组。组织内每个项目的的数据都相互隔离。在大多数情况下，您可以根据公司中的部门或者团队来设置项目。每个项目内，可创建 1 个或多个集群。查询或搜索操作仅针对特定项目。

## 角色和权限 {#roles-and-permissions}

不同角色的用户拥有不同级别的权限。权限决定了某一用户能否访问特性资源及进行特定操作。每种权限影响的范围由角色类型确定。

目前，有两种类型的组织角色:

- **组织管理员**：组织创建者的角色一般为组织管理员。同时用户被邀请进入组织时，也可以被分配组织管理员角色。组织管理员具有控制管理组织的所有权限，包括邀请或移除组织成员、修改现有组织成员角色或权限、[设置组织信息](./manage-organization-information)。

- **组织成员**：用户被邀请进入组织时，可以被分配为组织成员角色。组织成员只具有组织只读权限，可以查看组织设置，但无权邀请或移除组织成员或修改现有组织成员角色或权限 。

## 访问权限等级 {#access-level}

组织管理员具有所有项目级别的管理员权限。换言之，组织管理员具有控制和管理组织中所有项目的所有权限，包括查看和管理项目设置、项目集群和 Collection等。

但是，组织成员对于项目的访问级别取决于被授予的权限。组织管理员邀请组织成员时，可以选择成员可访问哪些项目以限制他们对组织下某一部分项目的权限。需要注意，当组织管理员授予组织成员特定项目的权限后，组织成员将成为这些项目的管理员，并拥有这些项目的所有权限，包括访问项目集群、Collection 及项目成员。

每个角色的权限如下所示。

| **权限**                  | **组织所有者**                                   | **组织成员** |
| ----------------------- | ------------------------------------------- | -------- |
| 组织设置信息                  | 是                                           | 只读       |
| 管理支付方式及账单               | 是                                           | 否        |
| 管理组织活动                  | 是                                           | 否        |
| 管理组织成员                  | 是                                           | 只读       |
| 离开组织                    | 是（每个组织必须至少拥有 1 个组织管理员，因此最后 1 名组织管理员无法离开组织。） | 是        |
| 创建项目                    | 是                                           | 否        |
| 项目特定数据（集群、Collection 等） | 是                                           | 仅限已授权的项目 |
| 集群数据和安全                 | 是                                           | 仅限已授权的项目 |
| 项目成员                    | 是                                           | 仅限已授权的项目 |
| 集群监控和告警                 | 是                                           | 仅限已授权的项目 |
| 项目设置                    | 是                                           | 仅限已授权的项目 |
