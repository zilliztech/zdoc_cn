# 添加组织成员

本教程介绍如何添加组织成员。

## 前提条件 {#prerequisites}

开始前，请先确保：

- 您已注册 Zilliz Cloud。如未注册，请先阅读[注册账号](./register-with-zilliz-cloud) 并完成注册。

- 您是组织管理员。更多用户角色和权限详情，请阅读[角色与权限](./roles-privileges) 。

## 操作步骤 {#steps}

1. 登录 [Zilliz Cloud 界面](https://cloud.zilliz.com.cn/login)。

1. 进入想要添加成员的组织。

1. 在左侧导航栏中，点击**组织成员**。您将看到组织用户列表。

1. 点击右上角的**邀请组织成员**。

1. 在**邀请用户**对话框中，输入**新用户**、选择**组织角色**和**项目权限（可选）**，然后点击**邀请**。

1. 相关参数请见下表。

| **参数**       | **描述**                                                                        |
| ------------ | ----------------------------------------------------------------------------- |
| **新用户**      | 邀请的用户的邮箱地址。您可以输入 1 个或多个邮箱地址。每输入 1 个地址，请按回车键检查邮箱地址是否有效。                        |
| **组织角色**     | 授予邀请的用户的组织角色。可以选择的角色包括：组织管理员、组织成员。更多角色和权限详情，请阅读 [角色与权限](./roles-privileges) 。 |
| **项目权限（可选）** | 授予用户有权访问的项目。默认情况下，组织成员有权限访问组织下的所有项目。如您希望组织成员只能访问特定项目，请在此字段中选择希望他们能够访问的项目。     |

:::info 说明

您可以一次邀请 1 个或多个相同角色的用户加入组织。每个组织最多可以包含 1,000 名成员。

:::

![org_members](/img/org_members.png)

## 结果 {#g-id-bold-result-g}

执行上述操作后，Zilliz Cloud 将向每个用户发送邀请电子邮件。在接受邀请之前，已被邀请的用户处于**待接受邀请**状态。一旦他们接受邀请，用户状态则会变为**已验证**表示用户已经加入了组织。您可以在组织用户列表页查看组织成员。
