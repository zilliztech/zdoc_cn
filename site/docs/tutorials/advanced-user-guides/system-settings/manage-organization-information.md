# 设置组织信息

Zilliz Cloud 允许用户查看并修改组织信息。

![organization-settings](/img/organization-settings.png)

## 查看当前组织信息 {#view-current-organization-information}

要查看当前组织信息，请从左侧导航栏中选择**系统设置**，并在**组织信息**下找到当前组织的 ID、名称和创建时间。

## 修改组织信息 {#modify-organizational-information}

当前， Zilliz Cloud 仅支持修改组织名称。如需更改组织名称，请单击**编辑**以打开**编辑组织名称**对话框，并在输入框中填写新名称。点击**确认**按钮以保存修改。如组织名称修改成功，您可在界面右上角看到提示信息“组织名称已修改”。
