# FAQ：资源规划

## 问答




### 什么是计算单元（Compute Unit, CU）? \{#what-is-a-compute-unit-cu}

计算单元（CU）是指用于提供向量检索、分析服务的一组硬件资源。可以将 CU 视为一个物理查询节点。

### 如何避免未使用的集群产生费用？ \{#how-can-i-avoid-expenses-on-unused-clusters}

建议您挂起未使用的集群以节省成本。您可随时按需恢复集群。
