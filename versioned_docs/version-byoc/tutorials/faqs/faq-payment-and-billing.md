### 在使用阿里云市场或亚马逊科技 Marketplace 订阅 Zilliz Cloud 服务时，是否可以使用阿里云云市场和亚马逊云科技 Marketplace 提供的优惠券抵扣 Zilliz Cloud 产生的相关费用？ \{#can-i-use-marketplace-credits-to-offset-zilliz-cloud-usage-when-subscribed-via-their-respective-marketplaces}

您在使用受支持的云服务提供商市场订阅 Zilliz Cloud 服务时，不能使用云服务提供商市场的优惠券抵扣您在 Zilliz Cloud 平台产生的费用。

在受支持的云服务提供商市场中订阅的 Zilliz Cloud 服务相对于该云服务提供商市场而言属于第三方服务。而各云服务提供商市场提供的促销或专属的优惠券仅用于订购该云服务提供商的自有服务，并不支持用于支付第三方服务。

具体而言，您可以参考各云服务提供商市场的相关条款。我们也摘录了相关条款的内容如下，供您参考：

- [阿里云云市场](https://help.aliyun.com/zh/user-center/user-guide/faq-about-coupons)：实物类以及云市场三方商品，云通信、域名、虚机、硬件类产品，以及通用优惠券/代金券不适用商品名单中的商品不包括在内。

- [亚马逊云科技 Marketplace](https://www.amazonaws.cn/promotional-terms/)：除非经我们授权，服务抵扣券不适用于抵扣关于非适用亚马逊云科技支持、亚马逊云科技Marketplace（由西云数据运营）、我们可能指定的任何其他服务，或者任何服务（如节省计划和预留实例）的任何预付费的任何费用或收费。

### 赠送的 ¥ 300 优惠券是否会过期？ \{#will-my-300-yuan-credits-expire-how-can-i-extend-the-expiration-of-the-free-credits}

在首次注册后，您可获赠¥ 300 优惠券，有效期 30 天。

### 数据被移至回收站后是否收费？ \{#is-data-in-the-recycle-bin-charged}

回收站中删除的数据不收取费用。

### Zilliz Cloud 组织冻结欠费后，为什么我在阿里云云市场充值了，Zilliz Cloud 组织仍未解冻？ \{#why-is-my-organization-still-frozen-even-if-i-recharge-my-aliyun-marketplace}

如您通过阿里云云市场订阅了 Zilliz Cloud服务，当您的阿里云账号余额不足时，阿里云将自动关闭您订阅的 Zilliz Cloud 服务，因此 Zilliz Cloud 账单界面也将不再展示阿里云云市场扣费作为您的支付方式。在阿里云充值后，如需解冻 Zilliz Cloud 组织，请先手动重新开通阿里云云市场扣费渠道。更多详情，请参考[订阅阿里云云市场](./subscribe-on-aliyun-marketplace#resubscribe)。
