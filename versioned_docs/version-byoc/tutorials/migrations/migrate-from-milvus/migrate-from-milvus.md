# 从 Milvus 迁移至 Zilliz Cloud

Milvus 是一个开源向量数据库，以高性能和易用性著称，广泛应用于管理大规模的向量数据。

Zilliz Cloud 通过提供托管的 Milvus 服务，简化了部署和维护的过程，使您可以轻松地将现有的 Milvus 部署迁移到云端。您可以通过连接 Milvus 实例或上传备份文件，将向量数据迁移到 Zilliz Cloud。
